# Práctica de Factorial: 

Este proyecto implementa el cálculo del factorial de un número en JavaScript.

# Descripción
El factorial de un número entero positivo "n" se define como el producto de todos los enteros desde 1 hasta "n".  

# Archivos
- index.html: Página base que enlaza el script.
- index.js: Contiene la lógica para solicitar un número, validar la entrada y calcular el factorial.

# Ejecución
1. Clona este repositorio.
2. Abre index.html en tu navegador.
3. Ingresa un número cuando se solicite.
4. El resultado se mostrará en la consola del navegador y en la página.
